1. The PSEO Scalable Core Formula (The "Variables")
   Programmatic SEO relies on scale. Instead of targeting single phrases, you target databases of modifiers combined with a root intent. In Denmark, housing search behavior is deeply localized down to neighborhoods (bydele) and educational hubs.

Your keyword generation architecture should map out three core pillars:

The Intent (Root): ledigt værelse, find roomie, bofællesskab, studiebolig, lejlighed til deling.

The Modifier (Location): {by} (Aarhus, Odense), {bydel} (Nørrebro, Trøjborg, Nordhavn), {postnummer} (2200, 8000), or {universitet} (AU, KU, SDU).

The USP (Value Add): gratis, uden abonnement, billig, direkte kontakt.

Example Scalable URL / Page Maps:
roomies.dk/ledige-vaerelser/{bydel} ➔ Targeting: "Ledigt værelse Nørrebro gratis"

roomies.dk/find-roomie/{postnummer} ➔ Targeting: "Find roomie 8000 Aarhus C"
